Atlanta Disc Golf Scorecard & Database
  • This site is used to record scores for local disc golf courses and
    • any number of players
    • any number of holes in any order
    • any teebox on any hole
  • Scores are saved to a database which means:
    • Scores saved using your phone can be viewed and edited at any time (including mid-round) using:
      • Another person's phone
      • Desktop computer
      • Tablet
    • Statistics can be compiled against all rounds played: Hole avg, min, max
  • Courses, teeboxes and pin locations are pre-programmed with corresponding pars
